tempo2 plugins driving you crazy? Try libstempo!

libstempo uses the tempo2 library to load a pulsar's tim/par files, and gives you easy Python access to the TOAs, the residuals, the timing-model parameters, the fit procedure, and more.

Requirements

  • Python 2.7 (no Python 3 yet...)
  • numpy
  • tempo2, of course (current version 2013.9.1)

Installation

  • You can install the latest release with pip install libstempo. That's all you need. Sweet, huh?
  • Otherwise, grab the git repository and do python setup.py install --prefix=.... The installer will look in likely places for the tempo2 headers and libraries, but you can also give it the tempo2 installation directory (the directory that contains include/tempo2.h and lib/libtempo2.a) using --with-tempo2=....
  • Since 2014/10/23, the master branch supports both the latest tempo2 release, and the latest CVS. (Or at least the changes that the maintainers are aware of. Open a ticket if you find a problem.)
